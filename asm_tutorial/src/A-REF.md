# Appendix A: Quick Reference

## C64 Color Palette

The C64 has a fixed palette of 16 colors, used by the VIC-II chip for borders, backgrounds, sprites, and character graphics.

| Value | Hex | Color |
|-------|-----|-------|
| 0 | $00 | Black |
| 1 | $01 | White |
| 2 | $02 | Red |
| 3 | $03 | Cyan |
| 4 | $04 | Purple |
| 5 | $05 | Green |
| 6 | $06 | Blue |
| 7 | $07 | Yellow |
| 8 | $08 | Orange |
| 9 | $09 | Brown |
| 10 | $0a | Light Red |
| 11 | $0b | Dark Grey |
| 12 | $0c | Grey |
| 13 | $0d | Light Green |
| 14 | $0e | Light Blue |
| 15 | $0f | Light Grey |

### Default Colors

On startup, the C64 uses:
- Border: 14 (Light Blue)
- Background: 6 (Blue)
- Text: 14 (Light Blue)
