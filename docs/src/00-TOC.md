# C64 Assembly Tutorial

A hands-on guide to learning 6502 assembly on the Commodore 64 using modern tools on Linux/WSL.

## Chapters

1. [Installation](01-INSTALL.md) - Setting up VICE and ACME on WSL/Ubuntu
2. [Hello World](02-HELLO.md) - Your first C64 program explained byte-by-byte
3. [Keyboard Input](03-INPUT.md) - Read keys, change colors, loops and branching
4. [Joystick Control](04-JOYSTICK.md) - Read joystick input with bit masking

## Appendices

- [Appendix A: Quick Reference](A-REF.md) - Color palette and common values
- [Appendix B: VIC-II Registers](B-VIC-II.md) - Graphics chip memory map
- [Appendix C: Exercise Solutions](C-SOLUTIONS.md) - Answers to chapter exercises
